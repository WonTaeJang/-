- markup: 자료들의 구조를 표현하기 위한 언어  
- tag: 
    - p: paragraph(본문)
    - h: heading
    - a: anchor
    - ul: unordered list
    - li: list item
    - ol: ordered list
- CSS: Cascading Style Sheet (스타일 보관 파일)
- css selector 우선순위가 있다. (더 우선적으로 적용됨)
    - style="" (1000점) // 스타일 속성
    - #id (100점)       // id selector
    - .class (10점)     // class selector
    - p (1점)           // tag selector
